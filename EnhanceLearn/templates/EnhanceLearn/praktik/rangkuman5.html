Ringkasan Materi
Pustaka Pillow

Pillow adalah pustaka Python yang digunakan untuk membaca, menampilkan, memodifikasi, dan menyimpan citra digital. Library ini merupakan pengembangan dari Python Imaging Library (PIL) dan sering digunakan dalam pemrosesan citra digital.

Mengubah Gambar Berwarna Menjadi Grayscale

Gunakan fungsi convert("L") untuk mengubah gambar RGB menjadi grayscale. Mode "L" digunakan untuk menghasilkan citra keabuan sehingga analisis citra menjadi lebih sederhana.

Mengubah Gambar Grayscale Menjadi Biner

Perubahan citra grayscale menjadi citra biner dilakukan dengan metode thresholding menggunakan point() dapat menentukan nilai ambang untuk membuat gambar biner.

Tranformasi Brightness

Gunakan fungsi ImageEnhance.Brightness() untuk mengatur tingkat kecerahan gambar.

Tranformasi Contrast

Gunakan fungsi ImageEnhance.Contrast() untuk meningkatkan atau menurunkan kontras gambar.

Tranformasi Negasi

Gunakan fungsi ImageOps.invert() untuk membalik intensitas warna gambar.

Tranformasi Thresholding

Gunakan fungsi point() untuk mengubah gambar grayscale menjadi gambar biner berdasarkan nilai ambang.

Tranformasi Logaritmik

Gunakan fungsi math.log() untuk memperjelas area gelap pada citra.

Tranformasi Power-Law (Gamma Correction)

Gunakan fungsi point(lambda p: int(255 * (p / 255) ** gamma)) untuk mengatur pencahayaan gambar menggunakan nilai gamma.

Tranformasi Histogram Equalization

Gunakan fungsi ImageOps.equalize() untuk meningkatkan kontras citra dengan meratakan distribusi histogram.

Tranformasi Histogram Specification

Gunakan histogram specification (histogram matching) untuk menyesuaikan distribusi intensitas suatu citra agar mendekati distribusi histogram citra referensi
