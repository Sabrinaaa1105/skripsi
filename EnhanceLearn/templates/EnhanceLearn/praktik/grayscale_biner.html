Pustaka Python Pillow

Bahasa pemrograman Python merupakan salah satu bahasa pemrograman yang digunakan dalam pemrosesan citra digital. Bahasa pemrograman Python banyak memiliki library (pustaka) untuk mengolah gambar. Pada materi ini, kita akan fokus menggunakan library (pustaka) Pillow. Pillow adalah pustaka Python yang dikembangkan dari Python Imaging Library (PIL) dan banyak digunakan untuk membaca, menampilkan, memodifikasi, serta menyimpan citra digital dalam berbagai format.

Mengubah Gambar Berwarna ke Grayscale

Dalam Pillow, proses mengubah gambar berwarna menjadi grayscale dilakukan menggunakan fungsi convert(). Fungsi tersebut digunakan untuk mengubah mode gambar ke format lain. Untuk grayscale, parameter yang digunakan adalah "L", yaitu mode citra keabuan. 

img = Image.open("gambar.jpg")
grayscale = img.convert("L")
grayscale.save("hasil_grayscale.jpg")

Pada contoh di atas, gambar berwarna dibuka menggunakan Image.open(), kemudian diubah menjadi grayscale dengan fungsi convert(“L”). Hasil akhirnya adalah citra yang lebih sederhana dan sering digunakan pada tahap awal pemrosesan citra digital, karena memudahkan analisis tanpa dipengaruhi informasi warna.

Mengubah Gambar Grayscale menjadi Gambar Biner

Proses perubahan grayscale ke biner dilakukan menggunakan metode thresholding. Pada metode ini ditentukan sebuah nilai ambang (threshold). Setiap piksel yang memiliki nilai intensitas lebih besar atau sama dengan threshold akan diubah menjadi putih, sedangkan piksel yang memiliki nilai lebih kecil dari threshold akan diubah menjadi hitam. Proses ini dapat dilakukan menggunakan fungsi .point(), yaitu fungsi yang digunakan untuk memetakan nilai setiap piksel berdasarkan aturan tertentu. Setelah itu, hasilnya dapat diubah ke mode biner menggunakan convert(“1”). 

img = Image.open("gambar.jpg").convert("L")
threshold = 128
binary = img.point(lambda p: 255 if p >= threshold else 0)
binary.save("hasil_biner.jpg")

Pada kode di atas, gambar terlebih dahulu diubah ke grayscale menggunakan convert(“L”). Selanjutnya setiap piksel dibandingkan dengan nilai ambang 128. Piksel yang nilainya 128 atau lebih akan menjadi putih, sedangkan yang kurang dari 128 akan menjadi hitam.
