SUBBAB 5: TRANSFORMASI INTENSITAS CITRA DENGAN PYTHON

Transformasi Intesitas Citra dengan Python

Library Pillow menyediakan berbagai fungsi yang dapat digunakan untuk melakukan transformasi intensitas citra, seperti pengaturan kecerahan (brightness), kontras (contrast), negasi (invers), thresholding, serta transformasi non-linear seperti logaritmik dan power-law (gamma). Fitur-fitur ini dapat diakses melalui modul seperti Image, ImageEnhance, dan ImageOps.
Sebagai contoh, ImageEnhance.Brightness digunakan untuk mengatur tingkat kecerahan citra, sedangkan ImageEnhance.Contrast digunakan untuk meningkatkan atau menurunkan kontras. Untuk operasi negasi, Pillow menyediakan fungsi ImageOps.invert yang secara langsung membalik nilai intensitas piksel. Operasi thresholding dapat dilakukan dengan memanfaatkan metode point() untuk memetakan nilai piksel berdasarkan ambang batas tertentu.

Brightness

img = Image.open("gambar.jpg")
brightness = ImageEnhance.Brightness(img)
hasil = brightness.enhance(1.5)

Pada kode di atas, gambar diubah menjadi grayscale lalu tingkat kecerahannya ditingkatkan menggunakan ImageEnhance.Brightness(). Nilai 1.5 membuat gambar menjadi 1,5 kali lebih terang dari gambar asli.

Contrass

img = Image.open("gambar.jpg")
contrast = ImageEnhance.Contrast(img)
hasil = contrast.enhance(1.5)

Pada kode di atas, gambar grayscale diproses menggunakan ImageEnhance.Contrast(). Nilai 1.5 digunakan untuk meningkatkan perbedaan antara area terang dan gelap sehingga detail gambar lebih jelas.

Invers

img = Image.open("gambar.jpg")
invert = ImageOps.invert(img.convert("L"))
hasil = ImageOps.invert(img)

Pada kode di atas, fungsi ImageOps.invert() digunakan untuk membalik nilai piksel. Piksel terang menjadi gelap dan piksel gelap menjadi terang sehingga menghasilkan citra negatif.

Thresholding

img = Image.open("gambar.jpg").convert("L")
threshold = 128
hasil = img.point(lambda p: 255 if p > threshold else 0)

Pada kode di atas, setiap piksel dibandingkan dengan nilai ambang (threshold) sebesar 128 menggunakan fungsi point(). Piksel di atas ambang menjadi putih, sedangkan piksel di bawah ambang menjadi hitam sehingga menghasilkan citra biner.

Logaritmik

img = Image.open("gambar.jpg").convert("L")
log = img.point(lambda p:int(255*math.log(1+p)/math.log(256)))

Pada kode di atas, transformasi logaritmik diterapkan pada setiap piksel menggunakan fungsi point(). Metode ini memperjelas detail pada area gelap tanpa membuat area terang terlalu dominan.

Power Law (Gamma Correction)

img = Image.open("gambar.jpg").convert("L")
gamma = 2.0
power_law = img.point(lambda p: int(255 * (p / 255) ** gamma))

Pada kode di atas, setiap piksel diproses menggunakan rumus gamma correction dengan nilai gamma = 2.0. Metode ini digunakan untuk mengatur kecerahan citra secara nonlinier, sehingga gambar menjadi lebih gelap atau lebih terang sesuai nilai gamma.
