SUBBAB 5: PERBAIKAN HISTOGRAM DENGAN PYTHON

Perbaikan Histogram Citra dengan Python

Library Pillow juga menyediakan berbagai fungsi untuk analisis dan manipulasi histogram citra. Histogram citra dapat diperoleh menggunakan metode histogram() yang menghasilkan distribusi frekuensi intensitas piksel pada citra. Distribusi tersebut dapat divisualisasikan dalam bentuk grafik menggunakan pustaka Matplotlib sehingga karakteristik citra, seperti tingkat kecerahan dan kontras, dapat diamati dengan lebih mudah.

Berdasarkan histogram yang diperoleh, proses seperti histogram equalization dapat dilakukan untuk meningkatkan kontras citra dengan meratakan distribusi intensitas menggunakan fungsi ImageOps.equalize(). Selain itu, terdapat pula teknik histogram specification (histogram matching) yang bertujuan menyesuaikan distribusi histogram suatu citra agar menyerupai histogram citra referensi tertentu. Proses ini umumnya memanfaatkan kombinasi Pillow dan NumPy untuk melakukan perhitungan distribusi histogram, fungsi distribusi kumulatif (Cumulative Distribution Function/CDF), serta pemetaan nilai intensitas piksel.

Untuk implementasi histogram dengan python ini, Pillow digunakan untuk membaca dan memproses citra, NumPy digunakan untuk membantu perhitungan histogram dan transformasi intensitas, sedangkan Matplotlib digunakan untuk menampilkan histogram dalam bentuk grafik sehingga distribusi intensitas piksel dapat dianalisis secara visual.

Histogram Equalization

img = Image.open("gambar.jpg")
img = Image.open("gambar.jpg").convert("L")
equali = ImageOps.equalize(img)

Pada kode di atas, gambar terlebih dahulu diubah menjadi grayscale menggunakan convert("L"). Selanjutnya fungsi ImageOps.equalize() digunakan untuk meratakan distribusi intensitas piksel sehingga kontras gambar meningkat dan detail menjadi lebih jelas.

Histogram Specification

img = Image.open("gambar.jpg").convert("L")
referensi = Image.open("referensi.jpg").convert("L")
hasil = histogram_specification(img, referensi)

Pada kode di atas, gambar utama dan gambar referensi diubah menjadi grayscale menggunakan convert("L"). Selanjutnya proses histogram specification dilakukan dengan menyesuaikan histogram gambar utama agar menyerupai histogram gambar referensi, sehingga karakteristik kecerahan dan kontras kedua gambar menjadi lebih mirip.
